Guía de Niveles para Estudiantes de Programación

Esta guía está diseñada para ayudar a estudiantes de programación a identificar su nivel de desarrollo y asignar puntos de complejidad a sus tareas. Cada estudiante puede seleccionar su nivel, y con base en esto, se le asignarán puntos específicos para evaluar el esfuerzo de cada tarea.

Cada nivel tiene un perfil asociado y una métrica de “Horas por Punto”, que representa el tiempo estimado que un estudiante en ese nivel requiere para completar un punto de esfuerzo.

Secuencia de Fibonacci para Story Points

La asignación de puntos de complejidad está basada en la secuencia de Fibonacci (0, 1, 2, 3, 5, 8, 13, etc.), que se usa comúnmente en la estimación ágil para reflejar el aumento de incertidumbre y complejidad en las tareas. A medida que aumenta la dificultad, se incrementan los puntos de esfuerzo, siguiendo esta secuencia.

Uso en Boards

Para asignar puntos a las tareas en los Boards:

  1. Selecciona tu nivel de la tabla de Niveles de Desarrollador. Esto determinará cuántas horas te llevará, en promedio, completar un punto de esfuerzo.
  2. Clasifica la tarea según la tabla de Complejidad de Tareas para asignarle un puntaje. Este puntaje representa la dificultad y el esfuerzo estimado en puntos.
  3. Multiplica los puntos por tu métrica de “Horas x Punto” para obtener una estimación personalizada del tiempo requerido para completar la tarea.

Este sistema estandariza la estimación de tareas y facilita la planificación de proyectos, adaptándose a diferentes niveles de experiencia en el grupo de estudiantes.
